# Proyecto: Sistema de Conversión de Unidades
Este proyecto es una aplicación de consola desarrollada en Python que cumple con los fundamentos de programación modular, tipos de datos y convenciones de nomenclatura. El programa permite convertir grados Celsius a Fahrenheit y determinar estados lógicos basados en los datos ingresados.

# Objetivos de la Tarea
- **Tipos de Datos:** Implementación de string (nombres), float (temperaturas), int (resultados redondeados) y boolean (estados lógicos).
- **Identificadores:** Uso de la convención snake_case para todas las variables y funciones.
- **Estructura de Carpetas:** Organización modular mediante paquetes y subcarpetas.
- **Control de Versiones:** Publicación del código fuente en un repositorio público de GitHub.

# Ejemplo de Funcionalidad
- **Entrada:** El usuario ingresa su nombre y una temperatura en Celsius (ej. 0).
- **Proceso:** El sistema calcula el equivalente en Fahrenheit y verifica si es punto de congelación.
- **Salida:** Muestra el resultado exacto, el redondeo a entero y la confirmación de estado.

# Conclusión
La realización de este proyecto permitió fortalecer el manejo de **tipos de datos fundamentales** en Python y la importancia de seguir estándares de industria como **PEP 8** para la nomenclatura de identificadores. 
Asimismo, la implementación de una **estructura modular** (separando modelos y servicios) facilita el mantenimiento y la escalabilidad del software. Finalmente, el uso de **GitHub** como herramienta de publicación consolida el flujo de trabajo profesional, permitiendo una gestión de versiones eficiente y transparente.
