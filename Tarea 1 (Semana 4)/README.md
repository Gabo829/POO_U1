Ejemplos del Mundo Real – Programación Orientada a Objetos (POO)
---------------------
Este repositorio contiene ejemplos sencillos en Python que aplican los conceptos fundamentales de la Programación Orientada a Objetos (POO) mediante situaciones del mundo real. Los ejemplos están diseñados para facilitar la comprensión de clases, atributos, métodos y la interacción entre objetos.

tienda.py – Sistema de Tienda (Carrito de Compras)
---------------------
Este programa modela una situación del mundo real donde una persona agrega productos a un carrito de compras. No se manejan pagos ni cálculos avanzados, ya que el objetivo es demostrar claramente el uso de la Programación Orientada a Objetos.

Objetivo

- Representar productos mediante una clase
- Permitir que una persona agregue productos a su carrito
- Mostrar los productos agregados

Clases utilizadas

- **Producto:** representa un producto con un nombre
- **Persona:** representa a una persona que puede agregar productos a un carrito

Conceptos de POO aplicados

- Definición de clases
- Uso de atributos
- Implementación de métodos
- Interacción entre objetos

reservas_hotel.py – Sistema de Reservas de Hotel
---------------------
Este programa representa un sistema sencillo de reservas de hotel, donde una persona puede reservar habitaciones y ver las reservas realizadas. El sistema es simple y está enfocado únicamente en la interacción entre objetos.

Objetivo

- Modelar habitaciones de un hotel
- Permitir que una persona realice reservas
- Mostrar las habitaciones reservadas

Clases utilizadas

- **Habitacion:** representa una habitación del hotel
- **Persona:** representa a la persona que realiza las reservas

Conceptos de POO aplicados

- Clases y objetos
- Atributos y métodos
- Relación entre clases

Conclusión
---------------------
Estos ejemplos demuestran cómo la Programación Orientada a Objetos permite modelar situaciones del mundo real de manera clara y organizada, facilitando la comprensión y mantenimiento del código.
