# Técnicas de Programación – Ejemplos Prácticos

Este repositorio contiene una colección de ejemplos diseñados para ilustrar las principales técnicas de programación orientada a objetos vistas en clase. Cada carpeta incluye un archivo Python con código funcional y comentado, pensado para facilitar la comprensión de los conceptos.

# Contenido del repositorio
## Abstracción
La abstracción permite modelar conceptos complejos ocultando detalles de implementación.
Ejemplo: clase abstracta Figura con métodos que deben implementar sus derivadas (Cuadrado, Circulo).

Ruta:
/abstraccion/abstraccion.py

## Encapsulación
La encapsulación protege los datos internos de un objeto mediante atributos privados y métodos controlados.
Ejemplo: CuentaBancaria con atributo privado __saldo.

Ruta:
/encapsulacion/encapsulacion.py

## Herencia
La herencia permite crear clases basadas en otras, reutilizando código y agregando funcionalidades.
Ejemplo: Perro y Gato heredan de Animal.

Ruta:
/herencia/herencia.py

## Polimorfismo
El polimorfismo permite usar un mismo método en distintas clases, cambiando su comportamiento según el tipo del objeto.
Ejemplo: método hablar() implementado en Perro y Gato.

Ruta:
/polimorfismo/polimorfismo.py

## Ejecución de los ejemplos
Cada archivo puede ejecutarse directamente con Python:

python nombre_del_archivo.py

Por ejemplo:

python abstraccion/abstraccion.py

## Objetivo del repositorio
Este repositorio fue creado como parte de la tarea de la asignatura Técnicas de Programación, con el propósito de:

- Aplicar los conceptos teóricos vistos en clase.
- Desarrollar ejemplos propios siguiendo buenas prácticas.
- Organizar y documentar el código de manera clara.
